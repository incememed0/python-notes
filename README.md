## Yorumlar

3 şekilde yorum oluşturabilirsin.

1. Kare işareti ile \#
2. Çift tırnak ile "
3. Tek tırnak ile '

```python
# tek satırlık yorumları '#' işareti ile oluşturuyoruz

""" 
daha uzun ve
satırlarca yorum 
yazmamız gerektiğinde tırnakları kullanabilirsin.
"""

''' veya
tek
tırnakla bile
yorum atabilirsin
'''
```
