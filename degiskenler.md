# Değişkenler

{% hint style="danger" %}
Değişken oluştururken dikkat etmen gereken 4 kural;  
1. Sayı ile değişken atayamazsın

2. - işareti kullanamazsın

3. Boşluk kullanamazsın

4. Türkçe karakterler kullanamazsın
{% endhint %}

```python
2degisken=5
degis-ken=5
degis ken=5
değişken=5
```
